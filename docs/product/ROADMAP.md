# Octopus 产品路线图

## 使用方式

路线图是状态仪表盘，不替代各版本执行文档。版本只能在退出条件全部满足后进入下一阶段。

| 版本 | 状态 | 主目标 | 关键依赖 | 退出条件 |
| --- | --- | --- | --- | --- |
| [v0.1](../releases/v0.1.md) | 内部里程碑 | Raw/Index 分离的 CLI MVP | 无 | 架构可行性已证明 |
| [v0.2](../releases/v0.2.md) | 内部里程碑 | 事务恢复和运行可观测 | v0.1 | 回滚、恢复、报告与校验可用 |
| [v0.3](../releases/v0.3.md) | 已完成 | 规模性能与检索可信度 | v0.2 | 大批量基线、搜索分批、证据链达标 |
| [v0.4](../releases/v0.4.md) | 已规划 | 安装和首次体验 | v0.3 | 80% 用户 10 分钟内获得首条有效结果 |
| [v0.5](../releases/v0.5.md) | 已规划 | 检索价值 Beta | v0.4 | Top-5 命中率≥80%，定位时间减少50% |
| [v0.6](../releases/v0.6.md) | 已规划 | 桌面端 Beta | v0.5、稳定服务 API | 核心工作流无需 CLI |
| [v0.7](../releases/v0.7.md) | 已规划 | Plugin 开发者预览 | v0.6、权限模型 | Package 与 Timeline 完成端到端验证 |
| [v0.8](../releases/v0.8.md) | 已规划 | 公开 Beta | v0.7、安装与升级链路 | 真实环境下关键指标达标 |
| [v0.9](../releases/v0.9.md) | 已规划 | Release Candidate | v0.8 | 无 P0/P1，所有 1.0 发布门槛通过 |
| [v1.0](../releases/v1.0.md) | 已规划 | 正式版 | v0.9 | 稳定、兼容、安装、升级和支持承诺生效 |

## 当前焦点

v0.3 已通过全部退出条件；下一活动版本必须从完成 v0.3 发布的 `main` 创建。

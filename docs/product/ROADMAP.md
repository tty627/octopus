# Octopus 产品路线图

## 使用方式

路线图是状态仪表盘，不替代各版本执行文档。版本只能在退出条件全部满足后进入下一阶段。

| 版本 | 状态 | 主目标 | 关键依赖 | 退出条件 |
| --- | --- | --- | --- | --- |
| [v0.1](../releases/v0.1.md) | 内部里程碑 | Raw/Index 分离的 CLI MVP | 无 | 架构可行性已证明 |
| [v0.2](../releases/v0.2.md) | 内部里程碑 | 事务恢复和运行可观测 | v0.1 | 回滚、恢复、报告与校验可用 |
| [v0.3](../releases/v0.3.md) | 已完成 | 规模性能与检索可信度 | v0.2 | 大批量基线、搜索分批、证据链达标 |
| [v0.4](../releases/v0.4.md) | 工程里程碑已完成 | 安装和首次体验 | v0.3 | 自动化安装链路通过；真人指标由产品负责人豁免且未声称达标 |
| [v0.5](../releases/v0.5.md) | 工程里程碑已完成 | 检索价值 Beta | v0.4 | 60/10 任务集 Top-5 90%/100%，解释契约通过 |
| [v0.6](../releases/v0.6.md) | 工程里程碑已完成 | 桌面端 Beta | v0.5、稳定服务 API | 桌面核心工作流与 Local API v1 契约通过 |
| [v0.7](../releases/v0.7.md) | 工程里程碑已完成 | Plugin 开发者预览 | v0.6、权限模型 | Package 与 Timeline 端到端及越权回归通过 |
| [v0.8](../releases/v0.8.md) | 工程里程碑已完成 | 公开 Beta 硬化 | v0.7、安装与升级链路 | 迁移回滚、诊断、兼容与规模门禁通过；真人/签名未声称 |
| [v0.9](../releases/v0.9.md) | 开发中 | Release Candidate | v0.8 | 无 P0/P1，契约冻结和 1.0 工程门槛通过 |
| [v1.0](../releases/v1.0.md) | 已规划 | 正式版 | v0.9 | 稳定、兼容、安装、升级和支持承诺生效 |

## 当前焦点

v0.8 已完成校验迁移回滚、本地脱敏诊断、两版本兼容矩阵、长路径/权限/同步盘保护和显式
性能回归审批。真实公开 Beta、签名与历史安装器练习未执行且未声称。当前重点转为 v0.9
的功能/Schema/API/Plugin 契约冻结、发布审计与 RC 工程门禁。

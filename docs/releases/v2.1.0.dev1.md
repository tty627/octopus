# Octopus 2.1.0.dev1

`2.1.0.dev1` 是面向学习与研究场景的未签名 Windows 开发预览版。它在
`2.1.0.dev0` 的多格式检索与可引用资料包基础上，补齐长任务反馈、AI 能力预检、
页面视觉分析和可靠导出。

## 功能摘要

- 新增统一任务中心：研究、资料包提案与导出任务可查看阶段进度、取消、失败后重试。
- 新增 AI Provider 能力探测和页面级视觉分析预检；发送页面图像前仍要求资料空间明确授权。
- 研究回答必须引用服务端提供的候选证据；无有效引用时安全降级为本地检索摘要。
- 资料包导出按请求隔离，避免并发导出的引用格式或“包含原文件”选项互相覆盖。
- 显式来源复核和导出前检查直接读取当前原文件哈希，不会用过期索引冒充最新状态。
- 资料包按模板把手动证据放入合理分组；归档前要求确认，并支持立即撤销归档。
- 首页研究问题会直接执行搜索；演示模式新建资料空间后会切换到独立的空空间。
- 导出下载带完整性校验和有效期；后台 Job 查询、轮询和取消均按资料空间隔离。
- 页面高亮预览会丢弃迟到响应并释放临时 Blob，切换证据时不会显示旧页面。

## 下载与安装

普通 Windows 11 x64 用户应下载 `Octopus-2.1.0.dev1-win-x64-setup.exe`。
不希望安装时可下载 portable ZIP，完整解压后运行 `Octopus.exe`。详细步骤和 SHA-256
校验方法见 [Windows 安装说明](https://github.com/tty627/octopus/blob/v2.1.0.dev1/docs/user/WINDOWS_INSTALLATION.md)。

## 发布验证

Tag 发布工作流在创建 GitHub Pre-release 前必须完成：

- Windows 与 Ubuntu、Python 3.12 的 Ruff、strict Mypy、pytest 和 85% 覆盖率门禁；
- ESLint、TypeScript、Vitest、Playwright Chromium 和前端生产构建；
- wheel、sdist、PyInstaller GUI/CLI、portable 包和安装器版本核对；
- Windows 静默安装、启动、卸载、重装，以及用户配置、原始资料和索引保留验证；
- 版本、变更记录、已知问题、Git commit、工件集合和 SHA-256 的 `release-audit`。

任一门禁失败时工作流不会发布或更新 Release。发布后的
`windows-install-validation.json` 和 `release-audit.json` 保存机器可读结果。

## 已知限制

- 本开发预览版未签名，Windows 可能显示 SmartScreen 提示；必须从本 Release 下载并核对 SHA-256。
- 当前发布范围是 Windows 11 x64；Windows 10、ARM64、网络盘、macOS 和 Linux 桌面端尚未形成发布承诺。
- Defender 干净机矩阵、正式 Authenticode 签名和真人研究用户验收仍是正式版门禁。
- AI 是可选能力；本地搜索、证据核对和资料包导出不依赖 API Key。

完整变更见 [CHANGELOG](https://github.com/tty627/octopus/blob/v2.1.0.dev1/CHANGELOG.md)。

# Octopus 规范文档说明

## 版本含义

`docs/specs/v1.0.1/` 中的 `v1.0.1` 是产品需求和文件格式规范的修订号，**不表示 Octopus 软件已发布 v1.0.1**。

当前各版本域独立管理：

- 产品开发版本：`0.4.0.dev0`。
- Leaf/FolderNode Markdown Schema：`octopus_schema: 0.2`。
- 配置、Manifest、Fingerprint、Prompt 和搜索缓存拥有各自的版本域。
- 规范修订号：当前目录为 `v1.0.1`。

详细规则见 [版本、分支与兼容性规范](../product/VERSIONING_AND_COMPATIBILITY.md)。
